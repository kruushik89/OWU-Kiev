Почему из 8500 языков программирования я выбрал JavaScript ?

Иван Иванов

Иван Иванов Веб-разработчик OktenWeb Педагог OktenWeb University на курсах Java Complex

Конечно, JS не единственный язык, который я сейчас использую. С другой стороны, с него все начиналось.

  • Во-первых, он настолько простой, что подходит даже для людей, которые в жизни не написали ни одной строчки кода.
  • Во-вторых, он бурно развивается.
  • В-третьих - повсеместно используется.

Нет ни одного современного браузера, который бы не использовал JavaScript, и это не единственная сфера его применения. На JS пишутся веб-приложения, серверные приложения, виджеты, прикладное ПО. Если вы, как и я, цените универсальность и простоту, записывайтесь на курс JavaScript Complex. Этот язык оправдает ваши ожидания.
